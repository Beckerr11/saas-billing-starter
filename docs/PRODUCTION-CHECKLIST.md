- [ ] staging configurado
- [ ] producao configurada

- [ ] logs estruturados

- [ ] branch protection validada
- [ ] dependencia critica sem alertas abertos

- [ ] testes automatizados verdes
- [ ] smoke test manual em staging
- [ ] changelog atualizado
